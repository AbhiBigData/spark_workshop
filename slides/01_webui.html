Spark Application's Web Console

  1. web UI is the web interface of a running Spark application for monitoring and inspecting Spark job executions in a web browser
  2. Enabled by default unless spark.ui.enabled used to disable
  3. Available at http://[driver]:4040
    • Use spark.ui.port to control the port

web UI Tabs

  1. Jobs shows status of Spark jobs
  2. Stages shows status of stages of all Spark jobs
  3. Storage shows RDD size and memory use
  4. Environment shows the environment
  5. Executors shows status of executors
  6. SQL shows status of structured queries
  7. Others feature-specific
    • Streaming or Thrift Server
  8. Switch to Mastering Apache Spark 2.0
